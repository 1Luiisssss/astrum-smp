// Normativa

REGLAS

01Respeto ante todo

El respeto es la base de la comunidad. Se prohíbe cualquier forma de acoso, discriminación, insultos o lenguaje ofensivo hacia cualquier miembro, tanto en el juego como en Discord.

02Sin trampas ni hacks

Queda estrictamente prohibido el uso de hacked clients, x-ray, dupes no autorizadas o cualquier exploit. Los mods técnicos aprobados están listados en Discord.

03Protección de builds

No destruyas, griefes ni robes las construcciones de otros jugadores. El robo de items sin permiso está castigado con ban permanente.

04PVP y zonas seguras

El PVP solo está permitido con consentimiento mutuo. El spawn y las zonas marcadas son áreas seguras. Las trampas que maten sin advertencia están prohibidas.

05Lag y granjas

Las granjas deben optimizarse para minimizar el impacto en el TPS. Granjas con más de 100 entidades activas deben tener kill automático. Consulta con los admins antes de construir granjas masivas.

06Decisiones de staff

Las decisiones del equipo de administración son definitivas. Puedes apelar en Discord con evidencia. El flame público al staff resultará en sanción adicional.

GUÍA DE OPTIMIZACIÓN

// JVM Flags
Flags de Java recomendadas
Usa Aikar's Flags con Fabric. Añade -Xmx4G -Xms4G -XX:+UseG1GC y el set completo de flags G1GC para reducir el garbage collection hasta un 60%.
JavaGCRendimiento
// Configuración de vídeo
Ajustes de vídeo óptimos
Distancia de renderizado: 10-12 chunks. Desactivar nubes y niebla. Graphics: Fast. Particles: Minimal. Smooth Lighting: Off. Biome Blend: 0. Mejora FPS en ~40%.
FPSGráficos
// Sodium + Iris
Mods de renderizado
Sodium es el reemplazo de OptiFine para Fabric. Combínalo con Lithium, Starlight e Indium. Ganas en media 3x más FPS sin sacrificar visibilidad.
SodiumLithiumFabric
// Red
Reducción de latencia
Usa cable Ethernet en lugar de WiFi. Cierra apps que consuman ancho de banda. Desactiva la VPN para jugar. Configura QoS en el router si es posible.
PingRed
// Granjas
Diseño anti-lag
Limita entidades por granja a menos de 100. Usa mob stacker cuando sea posible. Kill automático al alcanzar el límite. Evita granjas que generen lag de chunk loading.
TPSEntidadesAnti-lag
// RAM y CPU
Asignación de recursos
Asigna entre 4-8GB al juego (nunca más de la mitad de tu RAM total). Más RAM no siempre es mejor. Usa Java 21 LTS para mejor rendimiento con Fabric 1.21.1.
Java 21RAM
